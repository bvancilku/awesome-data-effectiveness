\section{How to Participate}

\begin{enumerate}
\item Think about your experiences with data projects.
\item On sticky notes, share your \textbf{successes} and \textbf{struggles} on the topics before you.
\item Save the QR Code or URL for later learning and conversation.
\end{enumerate}

\section{Successes and Struggles}

If you are joining us remotely, we asked participants to share successes and struggles in the following categories.

\begin{enumerate}
    \item Purpose and planning
    \item Storage and data quality
    \item Communication and visualization
    \item Reproducibility
\end{enumerate}

\section{Conclusion}

Our code is on GitHub at \url{https://github.com/bvancilku/awesome-data-effectiveness} . If you have an idea to add, please raise an issue or pull request. Thank you for visiting today!
